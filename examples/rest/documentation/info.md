# Show Camera information

Gets the information about the connected camera

**URL** : `/info`

**Method** : `GET`

## Success Response

**Code** : `200 OK`

**Content examples**

```json
{
  "id": "0xxx",
  "serialNumber": "B40IXXXXX",
  "vendorId": 11225,
  "productId": 33,
  "version": "1.0.X",
  "softwareVersion": "HuddlyIQ-X",
  "uptime": 2
}
```
