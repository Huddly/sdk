Detector

Current detector

  • Name: az_v20
  • Released: 20. Aug 2019

Changes

  • Significantly better at correctly detecting all labels.
  • More specialized towards use case. Only outputs person, head and chair label.
  • Slightly higher probability of detecting relfections.

Limitations

The detector is optimized to the cameras primary use case, and is therefore trained specifically towards small to medium sized conference rooms. This means that it is particulary good a detections up to approximately 5 meters, but less reliable outside of this range.

Output

Labels

Current labels the detector will output data for.

  • Person
  • Head
  • Chair

REST Api references

  • Start autozoom (genius framing) and detector : PUT /detector/start
  • Stop autozoom (genius framing) and detector : PUT /detector/stop
  • Get detections : GET /detector/detections
